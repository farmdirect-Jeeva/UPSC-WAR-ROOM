# 🇮🇳 UPSC War Room — IAS Command Centre

A comprehensive, all-in-one dashboard designed specifically for UPSC CSE aspirants. Track your preparation, manage time effectively, and stay motivated throughout your IAS journey.

## ✨ Features

### 🎯 Core Features
| Feature | Description |
|---------|-------------|
| **Focus Mode** | Full-screen distraction-free study mode with motivational quotes |
| **Pomodoro Timer** | Customizable study sessions (25/5/15 min) with session tracking |
| **Exam Countdown** | Dynamic countdown to your UPSC Prelims/Mains date |
| **Daily Motivation** | Curated inspirational quotes from great leaders |

### 📚 Syllabus & Progress Tracking
| Feature | Description |
|---------|-------------|
| **Syllabus Checklist** | Complete GS-I, GS-II, GS-III, GS-IV, CSAT, and Optional coverage |
| **Subject Progress** | Track progress across 8 core subjects with percentage updates |
| **Daily Tasks** | Priority-based task management (High/Med/Low) with completion tracking |
| **Weekly Study Hours** | Visual chart to log and monitor weekly study time |

### 🧘 Health & Wellness
| Feature | Description |
|---------|-------------|
| **Box Breathing** | Navy SEAL technique to reduce anxiety and improve focus |
| **Daily Routine** | Pre-loaded IAS topper routine (customizable) |

### 🤖 AI Study Friend
| Feature | Description |
|---------|-------------|
| **Claude AI Integration** | Ask UPSC-related questions and get instant, exam-focused answers |
| **Smart Suggestions** | Quick-access buttons for common UPSC topics |
| **Study Strategy** | Personalized advice for Prelims, Mains, and Optional preparation |

### 👥 Community & Organization
| Feature | Description |
|---------|-------------|
| **Study Groups** | Create/join virtual study circles |
| **Class Scheduler** | Track online classes with meeting links |
| **Customizable Routine** | Build your own daily schedule |

## 🚀 Quick Start

### Option 1: Direct Usage
1. Clone or download the repository
2. Open `upsc-warroom.html` in any modern browser
3. Start tracking your UPSC preparation!

### Option 2: GitHub Pages (Recommended)
1. Fork this repository
2. Go to Settings → Pages
3. Select main branch as source
4. Your dashboard will be live at `https://[username].github.io/upsc-war-room/`
